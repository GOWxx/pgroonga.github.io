# `pgroonga.libgroonga_version`パラメーター

## 概要

`pgroonga.libgroonga_version`パラメーターには使っているGroongaのバージョンが入っています。

このパラメーターは読み込み専用です。


## 構文

このパラメーターの値を表示する構文は以下の通りです。

```sql
SHOW pgroonga.libgroonga_version;
```

## 使い方

例です。

```sql
SHOW pgroonga.libgroonga_version;
--  pgroonga.libgroonga_version 
-- -----------------------------
--  8.0.1
-- (1 row)
```
